Sentinel Revenue Recovery™

AI-Powered Revenue Recovery & Revenue Leakage Detection

«Identify Lost Revenue. Recover More Opportunities. Strengthen Business Performance.»

Sentinel Revenue Recovery™ is an AI-assisted revenue intelligence platform designed to help organizations identify potential revenue leakage, recover missed opportunities, and improve financial performance through analytics, workflow automation, and actionable insights.

Instead of focusing only on generating new revenue, Sentinel Revenue Recovery helps businesses protect the revenue they have already earned—or could have earned.

---

Overview

Revenue can be lost for many reasons:

- Missed customer inquiries
- Unpaid invoices
- Subscription churn
- Scheduling gaps
- Billing errors
- Process inefficiencies
- Customer abandonment

Sentinel Revenue Recovery centralizes operational and financial data to help organizations detect these issues and prioritize recovery efforts.

---

Mission

Help organizations reduce preventable revenue loss through visibility, automation, and intelligent decision support.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
